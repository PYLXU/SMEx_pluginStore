# SMEx_pluginStore

[SimMusic扩展]适用于 SimMusic 的 第三方 扩展商店

## 扩展截图

![image](https://github.com/user-attachments/assets/57e61788-341b-4b38-b817-926b2c1cbc62)

## 安装教程

### 方式：直接安装

1. 下载此扩展到任意位置
2. 安装SimMusic后，打开「扩展」页面，将此扩展拖入扩展页面后点击「确认」按钮安装
